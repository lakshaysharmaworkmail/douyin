Douyin Live Scraper

Live status, viewers & profile data from Douyin URLs

Accepts full URL, short link, or just the room number

POST /api/scrape — Single URL

POST /api/scrape Content-Type: application/json { "url": "https://live.douyin.com/123456789" } // Response { "success": true, "data": { "status": "🟢 Live", "title": "Room title here", "nickname": "Streamer name", "totalViewers": "12000", "profileUrl": "https://www.douyin.com/user/...", "taskStatus": "✅ Done" } }

POST /api/scrape/batch — Up to 50 URLs

POST /api/scrape/batch Content-Type: application/json { "urls": ["https://live.douyin.com/111", "https://live.douyin.com/222"] } // Response { "success": true, "count": 2, "results": [ { "url": "...", "status": "🟢 Live", ... } ] }

GET /api/scrape?url= — Quick test

GET /api/scrape?url=https://live.douyin.com/123456789
